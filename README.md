Supermarket Sales Exploratory Data Analysis (EDA)
Project Summary:
An in-depth data exploration project that transforms raw retail transaction numbers into clean, visual business insights. It analyzes customer shopping behavior, branch performance, and payment trends to help optimize supermarket operations.

Key Highlights:

Data Cleansing & Sanity Checks: Inspected data integrity, verified data shapes, and handled missing values to ensure accurate analytical results.

Branch Performance: Compared sales traffic and volume across three major city branches (Yangon, Mandalay, and Naypyitaw).

Customer Profiling: Segmented transaction patterns by Gender and Customer Type (Members vs. Normal walk-ins) to identify the core target audience.

Payment Preferences: Visualized how customers prefer to pay (Cash, Credit Card, or E-wallet) to understand local transactional trends.

Data Visualizations: Used descriptive count plots and distribution graphs to turn flat data rows into presentation-ready insights.

Tech Stack & Libraries:
Pandas & NumPy
Matplotlib & Seaborn
